# v0.1.2 — 长期运行稳定性 + Tooltip + UI 对齐

**日期**：2026-04-17
**Release**：https://github.com/yosh3289/speedforce/releases/tag/v0.1.2

## 一句话概括

用户实际跑了一天多，暴露了 3 个 UX 问题：窗口失效、tooltip 截断、圆点不居中。都是代码里隐蔽的细节问题，必须真实使用才能发现。

## 主要修复

### 1. 长时间运行后窗口打不开（关键）
**问题**：顺利运行超过 1 天后，点"显示详情"和"设置"都没反应（没报错、窗口也不出来）。
**根因**：`DetailWindow.Show()` 每次都启一个新的 `go w.subscribe()` goroutine 跑 `for s := range sub { w.update(s) }` 循环。关窗时 `SetOnClosed` 只置 `w.win = nil`，没停止 subscribe 循环。每开关一次窗口就多一个 goroutine 永远活着，订阅 StateBus 并调 `fyne.Do(...)`。一天跑 1440 tick × N 个幽灵 subscribe 全都挤进 fyne.Do 队列，队列堆积导致新的 Show() 调度不进去。
**修复**：
- `detail.New()` 时启动**唯一一个**常驻 subscribe goroutine
- `Show()` 里不再启新的，只创建/显示窗口
- `update()` 内部已有 `if w.win == nil { return }` 守卫，窗口关闭时更新变无操作
- 窗口重开后，下次 state 到来时会正常填充新建的 widgets

### 2. Tooltip 截断
**问题**：多个服务同时 down 时（如 Claude Web / ChatGPT Web / Gemini Web），tooltip 最后一个被截成 "gem"。
**根因**：Windows `NOTIFYICONDATA.szTip` 硬上限是 127 字符（含 null 终止符）。原先格式：
```
SpeedForce ⚡
IP: 104.28.x.x (United States)
Services: 3/6 up
Down: Claude Web, ChatGPT Web, Gemini Web
```
约 100+ 字符，但 emoji 和 Unicode 字符在 UTF-16 占位不同，实际超限。
**修复**：
- 新格式更紧凑：
```
SpeedForce ⚡ 3/6 up
104.28.x.x US
✗ Claude Web
✗ ChatGPT Web
✗ Gemini Web
```
- 每个 down 服务独占一行（更易读 + 给字符预算让步）
- 国家名缩成 2-3 字母代码（"United States" → "US"），`countryCode()` 用单词首字母启发式
- 加 120 字符硬上限安全兜底，超过用 `…` 截断

### 3. 详情窗口圆点与 Label 不水平对齐
**问题**：服务行前的圆点看着比 Label 文本偏高。
**根因**：fyne `container.NewHBox` 按顶端对齐子元素，`canvas.NewCircle` 放 14×14 GridWrap 里占满从顶部开始，但 `widget.NewLabel` 有内部 padding（baseline 低于 14px）。
**修复**：换成 `container.NewBorder(nil, nil, container.NewCenter(dotCell), nil, label)`。Border 的 left 区域用 NewCenter 包裹让圆点在行高内垂直居中，Label 填充中心区域，两者 baseline 对齐。

## 经验教训

**隐蔽的 goroutine 泄漏**：subscribe-pattern 是常见 Go 错误模式。只有"生命周期绑定 goroutine"才安全——每次 Show() 都 start 新 goroutine 必须配 stop。或者用"整个应用生命周期一个 subscriber"模式（最终采用的方案）。

**Windows 原生 API 限制要认真读**：Windows tooltip 127 字符限制是 NOTIFYICONDATA 结构体字段大小决定的，不是 fyne/systray 的锅。这类平台级限制需要主动查询。

**UI 对齐是玄学**：fyne HBox 不自动垂直居中是常识，但只有真跑起来看才发现。container.NewBorder 的嵌套 NewCenter 是目前最干净的垂直居中方案。
