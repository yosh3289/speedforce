# v0.1.1 — Fyne 线程修复 + UI 打磨

**日期**：2026-04-16
**Release**：https://github.com/yosh3289/speedforce/releases/tag/v0.1.1

## 一句话概括

v0.1.0 本地试跑暴露了 fyne v2.7 严格线程检查导致窗口不显示、图标 emoji 字体缺失、设置不落盘、IP 地理拉不到、代理场景下延迟图标长期偏黄等五类真问题。v0.1.1 一次性修复。

## 主要修复

### 1. Fyne 主线程架构调整（关键）
**问题**：点"显示详情"后 cmd 窗口报 `*** Error in Fyne call thread, this should have been called in fyne.Do[AndWait] ***`，窗口不渲染。
**根因**：systray 占用主线程，fyne 事件循环从未启动，`fyne.Do` 只是排队但没人处理。
**修复**：
- 主线程跑 `fyneApp.Run()`
- `t.Run()`（systray）放 goroutine
- 创建一个 1×1 隐藏 master window 让 fyne 事件循环不因无窗口立即退出
- `DetailWindow.Show()` / `update()` / `SettingsWindow.Show()` 内部都包 `fyne.Do(...)`

### 2. 图标 emoji → canvas.Circle
**问题**：详情窗口服务行前的 🔵 能显示，🟡/🔴 却是方框带问号（tofu）。
**根因**：fyne 自带字体没收录 2019 年才加入 Unicode 12 的 large yellow/red circle。
**修复**：改用 `canvas.NewCircle(color)` + `container.NewGridWrap(14x14)` 组合画真彩色圆点，不依赖字体。

### 3. 托盘闪电图标换皮
**问题**：首版用户提供的像素艺术 PNG 染色出来的图标太糊。
**修复**：换成 Icons8 的 [flat-color-icons `flash_on`](https://github.com/icons8/flat-color-icons)（MIT 许可），三色变体在 `tools/make_icons.py` 里通过 Pillow polygon 绘制生成。

### 4. 设置持久化
**问题**：设置页保存后只在当次生效，退出重开又回到默认。
**根因**：main.go 的 Save 回调是 `func(newCfg *config.Config) error { return nil }` 空壳。
**修复**：真正调 `config.Save(cfgPath, newCfg)` 写回 YAML，语言改变时 `tr.SetLocale(newCfg.Language)` 同步切换。

### 5. IP 地理 API 换源
**问题**：IP 能拉到但国家/城市/ISP 长期为空（走代理时 ipapi.co 响应慢或被限流）。
**修复**：换到 `ip-api.com/json/{ip}`（免费、不限 IP、字段名不同，顺手改 `geoResp` 的 JSON tag 从 `country_name/org` 到 `country/isp`）。详情窗口 IP 信息空值时显示 `loading geo...` 而不是 `/ / /`。

### 6. "慢" 阈值调整
**问题**：走代理延迟常年 1-2s，托盘图标一直黄色，失去意义。
**修复**：`ComputeOverall` 和详情窗口的慢阈值从 1000ms 提到 **3000ms**。

### 7. 动态 tooltip
**问题**：托盘悬停只显示静态 "SpeedForce — 网络状态"，用户希望看到实时摘要。
**修复**：加 `Tray.UpdateTooltip(s core.State)`，主订阅 goroutine 每次状态更新时一并刷新。格式：
```
SpeedForce ⚡
IP: 104.28.x.x (United States)
Services: 6/6 up
```

### 8. Autostart 日志噪音
**问题**：每次启动打印 `autostart: The system cannot find the file specified.`（在 `AutoStart=false` 且从未写过注册表时）。
**修复**：调用前先 `IsAutoStart()` 比对，仅在状态不一致时写/删。

### 其他小修
- `detail/update()` 结尾加 `probesBox.Refresh()` 和 `spBox.Refresh()`，防止动态替换 Objects 后 UI 不刷新。
- `-H windowsgui` 链接标志（让本地 exe 也不再弹黑窗）。
